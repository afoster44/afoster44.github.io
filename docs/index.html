Keepr

Keepr was the final application Alex built while attending BoiseCodeWorks, a 13 week full stack software developer course. Keepr is a Pinterest replica and proved to be very challenging. When you first log in you are presented with a nice clean Bootstrap 4 Masonry style layout. From there you can proceed to log in and create your own vaults where you can then store any keep you want. A keep can be created by any user and be about anything you want!
